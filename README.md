# EverythingShop

A simple e-commerce application built with React as part of a FED assignment.

## Features

- Product listing fetched from API
- Product detail page with dynamic routing
- Discount calculation on product detail page
- Cart functionality with checkout flow
- Checkout success page that clears the cart
- Contact form with validation
- Responsive design using Bootstrap

## Tech stack

- React (CRA)
- React Router
- Zustand (state management)
- React Bootstrap
- Sass

## Getting started

```bash
npm install
npm start

```
